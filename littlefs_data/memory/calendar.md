# Calendar — next 14 days

Manually-maintained appointment list. Lango reads this in every morning
briefing and surfaces "📅 Today" + "Tomorrow" callouts. Anything beyond
14 days from today gets ignored to keep the file small.

**Format:** `YYYY-MM-DD HH:MM — <event>` (HH:MM optional for all-day items).
Use 24h time, PDT.

## How to use

Edit this file whenever you book or change something. The morning briefing
will pick up edits within the next cycle. If the file grows beyond 14 days
of future content, prune past entries — Lango ignores past items but the
clutter slows the LLM down.

Recurring weekly things (e.g. standups) belong here too as long as they
land in the 14-day window. Add and delete as needed.
